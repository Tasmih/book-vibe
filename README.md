# Book Vibe

A simple React app for managing books, including reading list and wishlist functionality.

## Features
- View all books
- Detailed book page
- Mark books as read
- Add books to wishlist
- Context API for state management

## Tech Stack
- React
- React Router
- Tailwind CSS
- Context API
